Pet Adoption Statistics

The website PetFinder.my is the most popular pet adoption website in Malaysia. With over 150,000 pets listed, their API has detailed records of adoption statistics. They have released a sample of this data and are hosting a competition on Kaggle that challenges programmers to generate an algorithm that will predict the likelihood of an animal being adopted. After all, anything is paw-sible!


The plots displayed for this project are an overview of some parameters that might impact adoption rates, which include age and gender. PetFinder.my uses an adoption speed classifier that will be used within most of the visuals; this classifier is used so see how quickly a pet has been adopted. A more advanced and fascinating way to dive deep within this dataset would be to use NLP tools to analyze the indivual pet descriptions or photo analysis of objective cuteness, but that would be some fur-ocious programming!
